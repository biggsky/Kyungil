• indexOf : 해당값이 있는 인덱스값 출력

  • find : find안에 값을 찾을때까지 돌고 해당값의 값을 출력

  • findIndex : findIndex안에 값을 찾을때까지 돌고 해당값의 인덱스값 출력

  • filter : arr[n] n번째에 값이 있으면 출력

  • map : 반환값들을 배열의 형태로 반환 해줌 +) true, false

  • forEach : 반복문 비슷함

  • join : 배열을 문자열로 변경시켜줌

  • split : 문자열을 배열로 변경.
    split 함수의 매개변수로 문자열 값을 자를 문자값을 넣어주면 된다.
    ',' 값을 매개변수로 전달하면 문자열에서 ,의 문자를 잘라서 배열의 형태로 변경시켜준다.
